Register the leerness Gate GitHub App

This pre-fills GitHub with the correct permissions and events. Deploy the Worker first to get its URL (see DEPLOY.md).

Org install? Use https://github.com/organizations/<org>/settings/apps/new as the form action.

After creating: copy the App ID, generate a private key, set a webhook secret, then set the three Worker secrets (DEPLOY.md step 3).

It will request: Checks read & write, Pull requests read, Contents read — subscribed to Pull request events.
